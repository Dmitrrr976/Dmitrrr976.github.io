Python: Циклы for и while

Цикл for

Для перебора элементов последовательности:

for элемент in последовательность: # выполняемый код

Пример со списком:

fruits = ["яблоко", "банан"] for fruit in fruits: print(fruit)
Вывод:
яблоко
банан

Цикл while

Выполняется пока условие истинно:

while условие: # код # важно изменять условие!
count = 0 while count < 3: print(count) count += 1
Вывод:
0
1
2

Функция range()

Генерация числовых последовательностей:

range(5) # 0,1,2,3,4 range(2,5) # 2,3,4 range(0,10,2) # 0,2,4,6,8
for i in range(3): print(i*2)
Вывод:
0
2
4

Оператор break

Немедленный выход из цикла:

for num in range(10): if num == 5: break print(num)
Вывод:
0
1
2
3
4

Важно!

В while без break убедитесь, что условие станет False, иначе цикл будет бесконечным.
